Moko

June-july 2024

back PROTOTYPE

Moko เว็บเพจสำหรับแสดงคินค้า ผลิตภัณฑ์เจลครีมบํารุงผิวชนิดไม่ล้างออก ให้ดูมีความธรรมชาติ และมินิมอล ตามสินค้า โดยนำเสนอเว็บไซต์เป็น Parallax Scrolling.


website
user flow
wireframe
prototype

problem

ต้องออกแบบเว็บเพจให้มีความหน้าสนใจ สำหรับขายสินค้าของเค้า โดยไม่ได้กำหนด User experience ว่าต้องการให้ผู้ใช้รู้สึกอย่างไร และ หัวข้อที่จะแสดงในเว็บไซต์ มีแค่ข้อมูลสินค้าพื้นฐาน ในไฟล์ PDF เท่านั้น ซึ่งต้อง แคปหน้่าจอ และแต่งรูปผ่าน Adobe photoshopแล้วปรับเป็น Element ที่จะใส่ในเว็บไซต์.

solution

ออกแบบให้เป็นเว็บเพจ แสดงเป็นหน้าเดี่ยว โดยเน้นการ Storytelling เล่าเรื่องเล่าสินค้า ผ่าน Animation และ Parallax Scrolling ตรง Landing Page ชูความโดดเด่นของสินค้า พร้อมใช้สีที่ตัดกัน ในแต่ละ Section

result

เว็บเพจมีความน่าสนใจ และเป็๋นที่ชื่นชอบต่อ Moko เพราะมีแบ่งสีที่ตัดกันในแต่ละ Section และ Parallax Scrolling ตรง Landing Page และ Animation ต่างๆ ใน Website ทำให้เพิ่มความน่าสนใจของสินค้ามากขึ้น
